OVERVIEW
The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence.
The project follows a structured approach commonly used in real-world analytics teams, covering data preparation, analysis, and visualization.

PROJECT OBJECTIVES
The main objectives of this project are:
    1.Clean and transform raw customer shopping data for analysis
    2.Model structured data for querying and reporting
    3.Use sql to analyze customer behavior, loyalty, and purchase drivers
    4.Build an interactive power bi dashboard to support data-driven decision making
    
DATASET

- Approximately 3,900 customer records
- Contains demographic, transactional, and behavioral data including:
  - Age and gender
  - Product category and item purchased
  - Purchase amount and discount usage
  - Subscription status and review ratings

WORKFLOW AND TOOLS

DATA PREPARATION, MODELING AND EXPLORATORY DATA ANALYSIS (PYTHON)

- Cleaned and transformed the raw dataset
- Handled missing values and inconsistent data
- Created derived features such as age groups and purchase frequency
- Exported a finalized dataset for downstream analysis

TOOLS USED
- Python
- Pandas
- NumPy
- Google Colab


DATA ANALYSIS (SQL)

- Imported cleaned data into a sqlite database
- Wrote business-focused sql queries to analyze customer behavior
- Analyzed customer segmentation and loyalty patterns
- Evaluated revenue contribution by category and demographics
- Studied discount usage and purchase impact

TOOLS USED
- SQLite
- DB Browser for SQLite
- SQL


VISUALIZATION AND INSIGHTS (POWER BI)

- Designed an interactive dashboard to visualize key business metrics
- Highlighted trends in customer behavior and revenue distribution
- Enabled filtering by subscription status, gender, category, and shipping type

TOOLS USED
- Power BI Desktop
